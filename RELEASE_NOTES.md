# CodeRainSaver 1.1.7

This public release includes the two supported installer builds:

- `CodeRainAppleSilicon-1.1.7-Apple-Silicon.pkg`: native Apple Silicon build for current macOS releases.
- `CodeRainIntel-1.1.7-Ventura-Intel.pkg`: Intel-compatible build for macOS Ventura 13.

Both installers place the screen saver in `/Library/Screen Savers` so it appears in System Settings after install.

This update restores animation in full-screen screen saver hosts while keeping the hidden/preloaded host throttle.

The `CodeRainCatalina` target remains experimental and is not included in the public release assets.
