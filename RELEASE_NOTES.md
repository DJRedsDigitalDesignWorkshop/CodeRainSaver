# CodeRainSaver 1.1.10

This public release includes the two supported installer builds:

- `CodeRainAppleSilicon-1.1.10-Apple-Silicon.pkg`: native Apple Silicon build for current macOS releases.
- `CodeRainIntel-1.1.10-Ventura-Intel.pkg`: Intel-compatible build for macOS Ventura 13.

Both installers place the screen saver in `/Library/Screen Savers` so it appears in System Settings after install.

This update stops the desktop-level Wallpaper host from rendering continuously while keeping preview, options, locked-session playback, and real screen-saver-level windows animated.

The `CodeRainCatalina` target remains experimental and is not included in the public release assets.
